# Installation

### 1. Install gulp-cli:

```sh
$ npm install gulp-cli --global
```

### 2. Install gulp and dependencies:
```sh
$ npm install gulp gulp-jade gulp-sass
```

### 3. Run gulpfile
```sh
$ gulp
```
